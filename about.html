Instructions About Penalty Calculation

  • Select the date on which book was issued
  • Select the date on which book was returned
  • Select the Country for which you want to calculate the penalty.

Some Important points to remember

  • Penality for Pakistan is Rs. 50/- per day
  • Penality for UAE is 5 DH per day
  • Penality is charged after 10 days.
  • Penality is calculated using only the business days. Weekends and Public Holidays are not included
